Como fazer migração de site WordPress para outro servidor

Neste artigo mostra o quão facil é fazer uma migração de um site WordPress para outra hospedagem

A primeira coisa a se fazer é ter certeza que você tenha acesso ao painel da hospedagem que está o site no momento e o painel da hospedagem onde o nosso site será migrado, pois você tera que acessar o banco de dados e o FTP do site.

Backup do Banco de dados

De inicio vamos fazer o backup do banco de dados WordPress, para isso basta acessar o cPainel de sua hospedagem, na sesão Banco de Dados deve se entrar no PhpMyAdmin e fazer o backup do banco de dados do site. Para isso basta selecionar todas as tabelas, na parte inferior do site onde está escrito "Com marcados:", basta escolher a opção "Exportar", uma tela ira abrir após isso basta clicar em "Executar" e o download se iniciara automaticamente.

Download dos arquivos do site

Agora que já temos a base de dados do site, vamos fazer a transferencia dos arquivos do site, você pode fazer isso via FTP ou pelo gerenciador de arquivos do Painel da hospedagem, neste caso eu irei mostrar como fazer isso utilizando via FTP com o Filezilla. Para isso, vamos criar um usuário FTP, devemos preecnher os campos com usuário e senha, após feito isso, abriremos o Filezilla e colocamos na parte superior o usuário FTP que criamos. Em "host", devemos colocar "ftp" mas o nome do seu site: "ftp.seusite", em "user", colocamos o nome do nosso usuario que criamos, que no meu caso é "admin" e no campo "password" colocamos a senha que criamos no usuario FTP. Feito isso, abrimos a pasta htdocs ou public_html que é onde os arquivos do nosso site está, após localizar a pasta onde fica os arquivos do wordpress, basta copiar eles e colocar em uma pasta qualquer da sua Area de Trabalho.
